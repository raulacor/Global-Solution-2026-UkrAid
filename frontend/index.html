Sobre

Inteligência de
Segurança Aberta

UkrAid é uma ferramenta de inteligência de linha de comando projetada para auxiliar civis em zonas de conflito ativo, centralizando dados críticos provindas fontes abertas em uma interface acessível.

Construída sobre a premissa de que informação salva vidas, a UkrAid agrega análises de imagens de satélite ao vivo, dados de eventos de conflito e coordenadas de recursos humanitários para que qualquer pessoa possa avaliar riscos e localizar ajuda.

A plataforma consulta quatro APIs em tempo real: ACLED para eventos de conflito geolocalizados, UkraineAlarm para monitoramento de alertas aéreos, OpenStreetMap para infraestrutura de refugiados e serviços de geocodificação — tudo sem exigir contas ou credenciais para funcionalidades principais.
